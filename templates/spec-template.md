## 使用说明

1. **创建新规格文档**:
   ```bash
   cp specs/spec-template.md specs/{编号}-{需求名称}/spec.md
   ```

2. **编写要点**:
   - 删除所有注释和说明文字
   - 专注于业务需求，不涉及技术实现
   - 保持简洁，2-3页即可

3. **后续步骤**:
   - 完成后使用 `/generate_plan` 生成技术方案（plan.md）
   - 再使用 `/generate_task` 生成任务清单（tasks.md）
